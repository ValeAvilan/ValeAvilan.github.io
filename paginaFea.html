HTML

Introducción


1. ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que se usa para crear páginas web.
No es un lenguaje de programación, es un lenguaje de estructura.
Es el estandar para estructurar y crear contenido de las páginas web

2. Etiquetas HEAD y BODY

HEAD es la parte invisible de la página.
CONTIENE:

  • título de la pestaña
  • configuración
  • enlaces a CSS

BODY es todo lo que el usuario ve
Se usan etiquetas detro para darle formato


Basic Tags


1. Explicar section, article, aside, header, footer tag

HEADER: Es la parte de arriba de la página.
FOOTER: Es la parte de abajo de la página. ej: derechos reservados
SECTION: Divide la página en secciones. ej: noticias
ARTICLE: Es contenido independiente. ej: un post o una noticia
ASIDE: Contenido secundario (como auncios o información extra)

CSS

Introducción


1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que se usa para dar estilo a una página web.

  • HTML es la estructura (este es el título)
  • CSS es el diseño (el título es rojo, grande y centrado)

2. ¿Para qué sirve CSS?

Sirve para:

  • Cambiar colores
  • Cambiar tamaños de letra
  • Acomodar elementos
  • Hacer la página responsive (que se vea bien en celular)
  • Hacer la página más bonita

3. ¿Cómo aplicar CSS a HTML?


  1. INLINE: se escribe dentro de la etiqueta HTML:
    -No se recomienda -> se usa para cosas rápidas

    ejemplo de INLINE

  2. INTERNAL: se pone dentro del HTML. Se pone en el head.
    -Mejor en INLINE -> No se recomienda para proyectos grandes

  3. EXTERNAL: se escribe en un archivo separado, creando un .css y conectándolo al archivo HTML
    Es la mejor forma y la que se usa en proyectos reales

Conceptos Importantes


a. ¿Qué es un SELECTOR?

Es lo que indica a qué elemento le quiero aplicar el estilo (este es un ejemplo en el que se aplica estilo al elemento p (afecta a todos los párrafos en el archivo .css))

b. ¿Qué es una REGLA CSS?

Es todo el bloque de estilo
Se compone de:

  • Selector → p
  • Propiedades → color, font-size
  • Valores → red, 20px

Basic CSS


Explicación:


Tag Selector


Selecciona todas las etiquetas HTML de un tipo. (ejemplo: p)


Class Selector

Selecciona elementos de una clase específica. (ejemplo: .titulo)

Tag and Class Selector

Selecciona una etiqueta específica con una clase. (ejemplo: p.importante -> solo afecta a la etiqueta p con clase importante)

Id Selector

Selecciona un elemento con ID ÚNICO (solo hay uno en toda la página/no se repite). (ejemplo: id="principal")

¿Qué es una CLASS y que es ID?

CLASS: se puede repetir. Es para estilos generales.
Por ejemplo: puede haber una clase subtitulos y aplicar a varios textos para que tengan el mismo estilo


ID: Es único. Se usa para darle estílo a algo específico.
Por ejemplo: id="titulo", si solo hay uno

Relación con CSS:

CSS usa:

  • . para clases
  • # para id

CSS BOX MODEL

Todo en HTML es una caja:

  • content → contenido (texto)
  • padding → espacio interno
  • border → borde
  • margin → espacio externo
